# xrossword
*A web app helps you solving crosswords.* [Check it out](https://xrossword.netlify.com)

## Get started

Install dependencies:

```bash
yarn install
```

...then start [Rollup](https://rollupjs.org):

```bash
yarn dev
```

To create a build for production:
```bash
yarn build
```
